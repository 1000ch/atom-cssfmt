## Install

```bash
$ apm install stylefmt
```

## Usage

- Packages > stylefmt > Format CSS
- Context Menu > Format CSS
- Command Palette (<kbd>⌘</kbd> + <kbd>shift</kbd> + <kbd>P</kbd>) > Format CSS
